# RDish

## 1. 概要

**RDish** は、外食時にメニューで出会う知らない料理名・食材名・調理法をすばやく調べられる Web アプリケーションです。

レシピを作るためのアプリではなく、**「外食で困らない人になる」** ことを目的とします。

ユーザーは、レストランのメニューや会話で出てきた知らない単語を検索し、「どんな料理か」「何が出てくるのか」「自分が頼んでよさそうか」を短時間で判断できます。

また、名前がわからない料理や食材についても、見た目・味・特徴から逆引きできます。

---

## 2. コンセプト

### メインコピー

> 外食で困らない料理図鑑

### サブコピー案

> メニューの「これ何？」をその場で解決する。

### 目的

- メニューに出てくる料理名・食材名・調理法を理解できるようにする
- 料理を注文するときの不安を減らす
- 知らない料理を覚えて、外食をより楽しめるようにする
- 料理名だけでなく、特徴から逆引きできるようにする

---

## 3. 想定ユーザー

### メインユーザー

- 外食で知らない料理名に出会うことが多い人
- ビストロ、イタリアン、フレンチ、ワインバー、エスニック料理店などに行く人
- 料理や食材に詳しくなりたいが、レシピを作る目的ではない人
- メニューを読んで、何が出てくるのか想像できるようになりたい人

### 利用シーン

- レストランで注文前に調べる
- 食事後に「あれは何だったのか」を復習する
- 旅行先や飲食店で知らない食材・料理に出会う
- 友人との会話で出てきた料理名を調べる
- 料理知識を少しずつ覚える

## 6. 主要機能

## 6.1 用語検索

ユーザーは料理名・食材名・調理法を検索できる。

### 検索対象

- 日本語名
- 英語名
- 原語表記
- 読み
- 説明文
- タグ
- 逆引きキーワード

### 表示する検索結果

検索結果には以下を表示する。

- 用語名
- カテゴリ
- ひとことで説明
- タグ
- 頼む判断ラベル

---

## 6.2 用語詳細ページ

各用語ごとに、外食時に判断しやすい情報を表示する。

### 表示項目

- 用語名
- 読み
- 英語名・原語名
- カテゴリ
- 国・地域
- ひとことで
- メニューで見たら
- 何が出てくるか
- 味・食感
- 頼む判断
- 苦手な人
- 似ているもの
- 関連用語
- タグ
- スコア（初心者向け・個性・重さ・辛さ）

---

## 6.3 逆引き検索

料理名がわからない場合に、特徴から検索できる。

### 逆引き対象

- 見た目
- 色
- 形
- 味
- 食感
- 調理法
- 食材
- 国・地域
- 似ている料理

### 実装方針

各用語に `reverseKeywords` を持たせてスコアリング検索する。

---

## 6.4 タグ検索

タグを使って用語を絞り込める。

### カテゴリ種類

- 料理 / 食材 / 調理法 / ソース / 香辛料 / チーズ / 野菜 / 肉 / 魚介 / デザート / 飲み物

---

## 6.5 頼む判断

料理の正確な定義だけでなく、注文時の意思決定を助ける。

### スコア項目

- `beginnerFriendlyScore`: 初心者向け度（1〜5）
- `uniquenessScore`: 個性の強さ（1〜5）
- `heavinessScore`: 重さ（1〜5）
- `spicinessScore`: 辛さ（0〜5）

---

## 6.6 似ているもの比較

用語同士の違いを比較する。

## 16. 注意点

RDish は料理の百科事典ではなく、外食時の意思決定アプリです。

説明文の優先順位:

1. 何が出てくるか
2. どんな味・食感か
3. 頼んでよさそうか
4. 似ているものとの違い
5. 詳細な歴史や専門知識
