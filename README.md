# RDish SPEC.md

## 1. 概要

**RDish** は、外食時にメニューで出会う知らない料理名・食材名・調理法をすばやく調べられる Web アプリケーションです。

レシピを作るためのアプリではなく、**「外食で困らない人になる」** ことを目的とします。

ユーザーは、レストランのメニューや会話で出てきた知らない単語を検索し、「どんな料理か」「何が出てくるのか」「自分が頼んでよさそうか」を短時間で判断できます。

また、名前がわからない料理や食材についても、見た目・味・特徴から逆引きできます。

---

## 2. コンセプト

### メインコピー

> 外食で困らない料理図鑑

### サブコピー案

> メニューの「これ何？」をその場で解決する。

### 目的

- メニューに出てくる料理名・食材名・調理法を理解できるようにする
- 料理を注文するときの不安を減らす
- 知らない料理を覚えて、外食をより楽しめるようにする
- 料理名だけでなく、特徴から逆引きできるようにする

---

## 3. 想定ユーザー

### メインユーザー

- 外食で知らない料理名に出会うことが多い人
- ビストロ、イタリアン、フレンチ、ワインバー、エスニック料理店などに行く人
- 料理や食材に詳しくなりたいが、レシピを作る目的ではない人
- メニューを読んで、何が出てくるのか想像できるようになりたい人

### 利用シーン

- レストランで注文前に調べる
- 食事後に「あれは何だったのか」を復習する
- 旅行先や飲食店で知らない食材・料理に出会う
- 友人との会話で出てきた料理名を調べる
- 料理知識を少しずつ覚える

---

## 4. アプリの方向性

### やること

- 料理名・食材名・調理法の検索
- 料理の簡潔な説明
- メニューで見たときの意味の説明
- 頼む判断の補助
- 見た目・味・特徴からの逆引き
- 似ている料理・食材との比較
- お気に入り・閲覧履歴
- クイズ・復習機能

### やらないこと

- レシピ投稿
- 調理手順の詳細解説
- カロリー管理
- 栄養管理
- 飲食店予約
- 食べログ的な店舗レビュー
- 本格的な料理史データベース

### 検索対象

- 英語名
- 原語表記
- 読み
- 説明文
- タグ
- 逆引きキーワード

### 検索例

- コンフィ
- ロマネスコ
- ブッラータ
- タブレ
- ビリヤニ
- エスカベッシュ
- アヒージョ
- フムス
- ハリッサ

### 表示する検索結果

検索結果には以下を表示する。

- 用語名
- カテゴリ
- ひとことで説明
- タグ
- 頼む判断ラベル

```txt

タグ: 肉料理, 油, 低温調理, フランス
頼む判断: しっとり肉料理が好きならおすすめ
```

## 6.2 用語詳細ページ

各用語ごとに、外食時に判断しやすい情報を表示する。

### 表示項目

- 用語名
- 読み
- 英語名・原語名
- カテゴリ
- 国・地域
- ひとことで
- メニューで見たら
- 何が出てくるか
- 味・食感
- 頼む判断
- 苦手な人
- 似ているもの
- 関連用語
- タグ
- 覚え方メモ

### 詳細ページ例: コンフィ

```md
# コンフィ

## ひとことで
低温の油脂でじっくり加熱したフランス料理の調理法。

## メニューで見たら
肉や野菜を油でじっくり火入れした料理の可能性が高い。
特に「鴨のコンフィ」は定番。

## 何が出てくるか
- 鴨肉
- 豚肉
- 鶏肉
- にんにく
- 野菜

## 味・食感
- しっとり
- ほろほろ
- 脂のコクがある

## 頼む判断
ガッツリした肉料理や、しっとりした食感が好きならおすすめ。

## 苦手な人
脂っぽい料理が苦手な人には少し重い場合がある。

## 似ているもの
- アヒージョ
- オイル煮
- 低温調理
```

### 詳細ページ例: ロマネスコ

```md
# ロマネスコ

## ひとことで
黄緑色で渦巻き状の見た目をした、ブロッコリーやカリフラワーに近い野菜。

## メニューで見たら
サラダ、グリル、付け合わせで出てくることが多い。
見た目は特徴的だが、味は比較的食べやすい。

## 何が出てくるか
- 黄緑色の幾何学的な野菜
- 小房に分けられた状態
- グリルや茹で野菜

## 味・食感
- ブロッコリーに近い
- カリフラワーより少し甘みがある
- コリコリ、ほくほく

## 頼む判断
野菜が苦手でなければ安心して頼みやすい。

## 似ているもの
- ブロッコリー
- カリフラワー
- カリフローレ
```

## 6.3 逆引き検索

料理名がわからない場合に、特徴から検索できる。

### 入力例

```txt
白くて中がとろっとしたチーズ
```

結果:

- ブッラータ
- モッツァレラ

```txt
緑色でブロッコリーみたいな渦巻きの野菜
```

結果:

- ロマネスコ

```txt
魚を揚げて酸っぱい液につけた料理
```

結果:

- エスカベッシュ
- 南蛮漬け

### 逆引き対象

- 見た目
- 色
- 形
- 味
- 食感
- 調理法
- 食材
- 国・地域
- 似ている料理

### 実装方針

MVP では、各用語に `reverseKeywords` を持たせて簡易検索する。

将来的には、LLM API またはベクトル検索を利用して自然文検索に対応する。

## 6.5 頼む判断

RDish の重要機能。

料理の正確な定義だけでなく、注文時の意思決定を助ける。

### 表示項目

- おすすめする人
- 注意した方がいい人
- 味の方向性
- 食べやすさ
- クセの強さ
- 辛さ
- 重さ

### ラベル例

```txt
安心して頼みやすい
好みが分かれる
辛いもの好き向け
香草が苦手なら注意
脂っぽい料理が苦手なら注意
野菜が苦手でなければ安心
```

## 6.6 似ているもの比較

用語同士の違いを比較する。

### 例: コンフィ / アヒージョ / オイル煮

| 用語 | 違い |
|---|---|
| コンフィ | 低温の油脂でじっくり加熱するフランス料理の調理法 |
| アヒージョ | にんにくとオリーブオイルで煮るスペイン料理 |
| オイル煮 | 油で煮る料理全般を指す広い表現 |

### 例: ブッラータ / モッツァレラ / リコッタ

| 用語 | 違い |
|---|---|
| ブッラータ | 外側はモッツァレラ、中はクリーム状でとろっとしている |
| モッツァレラ | 弾力のある白いフレッシュチーズ |
| リコッタ | ほろほろした軽い食感のチーズ |

## 6.9 クイズ機能

MVP 後の追加機能。

```txt
黄緑色で、ブロッコリーやカリフラワーに近い渦巻き状の野菜は？
```

```txt
ロマネスコ
```

## 12. UI 方針

## 12.1 デザイン方針

- スマホで使いやすい
- 飲食店の席で素早く見られる
- 文字が読みやすい
- 検索導線を最優先
- 料理写真がなくても成立する
- 難しい料理知識より、注文判断を優先する

## 12.2 トーン

- 短く
- わかりやすく
- 外食目線
- 専門用語を避ける
- ただし雑すぎない

### 悪い例

```txt
コンフィとは、主に食材を保存する目的で脂肪中にて低温で長時間加熱するフランスの伝統的調理法であり...
```

### 良い例

```txt
油でじっくり火入れした、しっとり系の料理。鴨肉でよく出ます。
```

## 13. SEO 方針

各用語ページは検索流入を狙える。

### ページタイトル例

```txt
コンフィとは？外食メニューで見たときの意味・味・頼む判断 | RDish
```

```txt
ロマネスコとは？ブロッコリーとの違い・味・食べ方 | RDish
```

### description 例

```txt
コンフィとは、油でじっくり火入れしたフランス料理の調理法です。外食メニューで見たときに何が出てくるのか、味や頼む判断をわかりやすく解説します。
```

### URL 例

```txt
/dishes/confit
/dishes/romanesco
```

- Chrome
- Safari

- 文字サイズを十分に大きくする

- ボタンにはラベルを付ける
- キーボード操作に対応する

## 20. 注意点

RDish は料理の百科事典ではなく、外食時の意思決定アプリです。

そのため、説明文では次の順番を優先する。

1. 何が出てくるか
2. どんな味・食感か
3. 頼んでよさそうか
4. 似ているものとの違い
5. 詳細な歴史や専門知識

専門的に正確であることは大事だが、最初の価値は **「メニューを見て困らない」** ことに置く。
